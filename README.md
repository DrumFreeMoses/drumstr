# Drumstr

A modular, mobile-first, global virtual group drumming network.

## Project Structure

- `apps/web/` — Web app frontend (excluded for now)
- `packages/shared/` — Shared libraries and utilities
- `specs/` — Project specifications
- `docs/` — Documentation
